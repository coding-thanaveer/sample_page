Git

Git is an version control tool

    One time setup of git:

  1. git config --global user.name "Your name" => tells git this is my username
  2. git config --global user.email "Your email" => tells git this is my email

    3. Basic commands

  3. git init => intialize the local git repository
  4. git add filename => stage the file contents for the next commit. here stage represents the choosing the file contents for the next commit.
  5. git commit -m "message for a file commit"
